# パスワードリセット機能

## 概要
ログイン画面から「パスワード忘却時のリセットはこちら」リンクを辿り、メールアドレス入力→メール送信→トークン付きURLでパスワード変更するフロー。

## 環境変数（必須）
メール送信のために以下を `.env.local` に設定してください。

| 変数名 | 説明 |
|--------|------|
| `GMAIL_USER` | 送信元メールアドレス（デフォルト: yokohama.aoba.pickles@gmail.com） |
| `GMAIL_APP_PASSWORD` | Gmailアプリパスワード（2段階認証有効時は要取得） |
| `NEXT_PUBLIC_APP_URL` | アプリのベースURL（例: https://your-app.vercel.app） |

## トークン有効期限
30分

## セキュリティ
- メールアドレスが存在しない場合：何もしない（エラーも出さない）
- メール存在時も「入力されたメールアドレスが登録されている場合、リンクをお送りしました」と表示し、列挙攻撃を防止
