# 会員登録フォーム：キャンセルボタン追加およびUI調整仕様

## 1. キャンセルボタンの追加

### ● 追加位置
- 「新規会員登録申請」ボタンの左側に配置。

### ● 動作
- クリック時：  
  `router.push('/members/login')`  
  → ログイン画面へ遷移。
- フォーム送信や状態クリアは行わない。

---

## 2. ボタンサイズの等分

### ● レイアウト変更
- 最下部のボタンエリアを以下のスタイルに変更：
  - `display: flex`
  - `gap: 16px`

### ● ボタン幅
- キャンセルボタン、新規会員登録申請ボタンの両方に `flex: 1` を指定し、幅を等分にする。

### ● 申請ボタンの調整
- `width: 100%` を廃止。
- 角丸を `borderRadius: '8px'` に統一（他ページと合わせる）。

---

## 3. キャンセルボタンのスタイル（他ページと統一）

`members/profile/edit` のキャンセルボタンと同じデザインに統一。

### ● スタイル仕様
```ts
backgroundColor: 'transparent'
color: '#888'
border: '1px solid #444'
padding: '14px'
borderRadius: '8px'
fontSize: '1rem'
```
